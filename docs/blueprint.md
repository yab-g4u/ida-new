# **App Name**: IDA: Your AI Health Ally

## Core Features:

- Medicine Search & Simplification: Search for medications by name or symptom and receive simplified instructions in Amharic, Oromo, and English. Uses the Gemini API to simplify complex medical text using system prompt as a tool.
- AI Health Assistant: A conversational AI health assistant that provides real-time, grounded information in Amharic, Oromo, and English, powered by the Gemini API with Google Search tools for citations.
- My QR Info: Securely store and retrieve vital medical information (allergies, prescriptions, etc.) using Firestore, generating a scannable QR code for emergency access.
- Trilingual Support: Full application support for Amharic, Oromo, and English languages, ensuring accessibility for a diverse user base.
- Voice Input: Ability to verbally query about medication in all supported languages, particularly Amharic.
- Secure Authentication: Initialize Firebase Authentication immediately to secure the 'My QR Info' and state management features.

## Style Guidelines:

- Primary color: Light cyan (#B0E2FF), evoking feelings of trust and calm, drawing inspiration from healthcare environments. It creates a reassuring digital space.
- Background color: Pale cyan (#E5F5FF), nearly white, provides a clean and non-intrusive backdrop, ensuring readability and focus on key content.
- Accent color: Soft sky blue (#87CEEB), is utilized for interactive elements such as buttons and links, drawing attention without overwhelming the senses.
- Headline font: 'Belleza' (sans-serif), will provide personality aligned to health, art, and design. Body font: 'Alegreya' (serif) to make long passages of text readable.
- Simple, clear icons for navigation and features, prioritizing ease of understanding for elders and those with limited health literacy.
- Mobile-first, fully responsive design with a clear, uncluttered layout, ensuring optimal viewing and interaction on all devices. High contrast to assist elderly vision.
- Subtle animations and transitions to guide users and provide feedback, enhancing the user experience without being distracting.
